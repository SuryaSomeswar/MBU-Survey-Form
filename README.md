# MBU Survey Form

A simple responsive web survey form built using basic HTML to collect feedback from users about the MBU platform.

## Features

- Collects user name, email, and optional age.
- Dropdown to select the user's current role (Student, Full-time job).
- Radio buttons to rate how likely the user is to recommend the platform.
- Checkboxes to choose what areas need improvement (projects, coding challenges).
- Textarea for additional comments or suggestions.
- Basic HTML structure ready to be styled with CSS.

## Technologies Used

- HTML5
